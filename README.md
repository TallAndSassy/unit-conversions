## Installation

You can install the package via composer:

```bash
composer require tallandsassy/unit-conversions
```

## Usage

``` php
$skeleton = new Tallandsassy\UnitConversions();
echo $skeleton->echoPhrase('Hello, Tallandsassy!');
```
